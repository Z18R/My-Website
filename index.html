Hardware/Software

A plastic bottle operated reverse vending machine is a machine which accepts plastic bottles and convert it into either water or money. Aside from plastic bottles, it also accepts coins as a payment for the water supply.

This type of vending machine will be using image processing to detect plastic bottles and a weighing sensor which will weigh plastic bottles and convert it into its equivalent amount.

The machine will decide how much it will dispense depending upon the weight and quantity. This machine is one of the platforms which can be used to manage and recycle plastic bottles.
